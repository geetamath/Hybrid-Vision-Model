# Hybrid Vision Model: Haar + YOLO for Robust Detection

## Overview
This project integrates Haar Cascade (for face detection) and YOLO (for object detection)
into a Flask web application.

## Features
- Face detection using OpenCV Haar Cascades
- Object detection using YOLO v4 Tiny
- Web interface for image upload and detection

## Tech Stack
Python, OpenCV, Flask, YOLO, HTML, CSS, JavaScript

## Run
pip install -r requirements.txt
python app.py
